Three.js 音频和场景关联
• 位置音频系统（PositionalAudio）
• 5种不同音频源（钟声/鼓声/钢琴/贝斯/弦乐）
• 距离衰减和音量控制
• 监听器跟随相机

核心知识点：

1. 位置音频
   - AudioListener 监听对象
   - PositionalAudio 3D音源
   - 距离衰减模型

2. Web Audio API
   - Oscillator 振荡器
   - GainNode 增益节点
   - 频率和音量控制

3. 音频参数
   - rolloffFactor 衰减系数
   - refDistance 参考距离
   - distanceModel 距离模型
