Three.js 音乐可视化
• 内置音频合成器（8种不同波形）
• 4种可视化模式（柱状图/波浪/环形/球体）
• 频率分析和实时渲染
• 动态视觉效果

核心知识点：

1. Web Audio API
   - AudioContext 音频上下文
   - AnalyserNode 分析器
   - OscillatorNode 振荡器

2. 频率分析
   - fftSize 快速傅里叶变换
   - getByteFrequencyData 获取频谱
   - 频率数据映射视觉

3. 视觉效果
   - 柱状图频谱
   - 波浪动画
   - 环形分布
   - 球体可视化
